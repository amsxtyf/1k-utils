# 1k-utils

A toolkit that contains common features used in the day-to-day development of the Web。

[docs](https://kevily.github.io/1k-utils/)

## npm install

```shell
npm install 1k-utils
```

## yarn instatll

```shell
yarn add 1k-utils
```

## Usage

```javascript
import { storage } from '1k-utils'
storage('local').set('1', { key: '1' })
```
